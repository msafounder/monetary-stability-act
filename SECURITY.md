# Security Policy

## Reporting a Vulnerability

If you discover a security issue related to the Monetary Stability Act (MSA) framework, please report it privately.

Do NOT disclose vulnerabilities publicly before a fix or clarification is issued.

## Scope

This repository contains:
- Framework documentation
- Whitepaper specification
- Future technical scripts

Any implementation-related security concern should be documented via a private report.

## Disclaimer

MSA is currently in the framework phase.
No production token has been deployed at this stage.
